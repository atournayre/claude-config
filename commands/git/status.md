# Statut Git

Lire les `Fichiers` et exécuter les `Commandes` puis résumer l'état actuel du dépôt git.

## Commandes

- Statut actuel : !`git status`
- Diff actuelle : !`git diff HEAD origin/main`
- Branche actuelle : !`git branch --show-current`
